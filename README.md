# The is Massively
### Challenge 2 - Reproduction de l'index du site The is Massively


![Zozor](/header.png)
---
* Votre mission ~~si vous l'acceptez~~ est de reproduire l'index du site internet [The is Massively](https://html5up.net/massively) !
* Les **Hovers des images sont en bonus** si vous avez le temps de les réaliser, dans ce cas, n'hésitez pas à les faire, c'est fun quand on maîtrise la bête.
* Le **Header est également un gros bonus, comme c'est du JS.** Autrement laissez le en forme comme sur la photo ci-dessus.
* Pour le reste, tout est à créer, **soyez vigilant sur les détails**, comme le changement de couleur au passage de la souris ou au clic.
* Le framework **Boostrap est à utiliser**.
* **Page Ruler** est un addon de règle pour chrome.
* Les images sont fournies.
